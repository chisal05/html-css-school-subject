프로그래밍 용어 정의

다음은 자주 사용되는 프로그래밍 용어들입니다.

  • HTML: 웹 페이지의 구조를 정의하는 마크업 언어입니다.
  • CSS: 웹 페이지의 스타일을 지엉하는 스타일 시트 언어입니다.
  • JavaScript: 웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어입니다.
  • API: 애플리케이션 간의 상호 작용을 위한 인터페이스입니다.

프로그래밍 언어의 종류

프로그래밍 언어는 크게 두 가지로 나뉩니다.

  • 저급 언어
