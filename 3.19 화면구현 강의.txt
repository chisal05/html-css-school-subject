텍스트 태그
> <h1>~<h6> : 제목을 표현하기 위해 사용

br 태그와 &nbsp;
> <br> : 줄바꿈할 때 사용  : 엔터키와 비슷한 의미
   &nbsp; : 공백을 나타낼 때 사용 : 스페이스바와 비슷한 의미

특수 문자에 대한 html 표기 방식
> &nbsp; : 공백을 나타냄
   &lt; : " < "
   &gt; : " > "
   &amp; : " & "
   &quot; : " " "
   &copy; : " ⓒ "

p 태그
> <p></p> : 단락을 구분할 때 사용

b 태그
> <b></b> : 글씨체 굵게 처리 (글씨 bold 처리)

i 태그
> <i></i> : 글씨체 기울여 처리 (글씨 italic 처리)

순서 없는 목록 나타내기
> <ul><li></li></ul>
	ex) ● 순서1
                 ● 순서2
                 ● 순서3

순서 있는 목록 나타내기
> <ol><li></li></ol>
	ex) 1. 순서1
                  2. 순서2
                  3. 순서3

a 태그
> <a href=""></a> : 링크 걸기 (웹 페이지에서 메뉴, 베너 등을 클릭하면 지정된 페이지로 이동 (하이퍼링크를 건다고 생각하면 편함))
    링크는 a 태그를 사용하여 표현

img 태그
> <img src="" alt="" width="" height=""> : 파일에 저장한 이미지 띄우기
   src에는 지정한 파일 이름을 쓰고, alt에는 지정하고 싶은 이름을 쓰면 됨.
   width는 가로 길이 지정, height는 세로 길이를 지정할 때 사용. 길이 단위는 px을 사용.
	ex) <img src="image.jpg" alt="사진">

주석문
> 웹디자이너 또는 프로그래머가 이해하기 쉽게 코드 설명을 넣는 것
   실행화면에는 해당 글자는 뜨지 않는다.
	ex) <!-- 이 말은 주석처리에 대한 말입니다. -->


header 태그, nav 태그, section 태그, footer 태그

header : 도입부에 해당하는 콘텐츠나 네비게이션 링크의 집합 등과 같은 정보를 포함함.

> 하나 이상의 제목 요소(h1 ~ h6)
   로고(logo)나 아이콘(icon)
   저자(author) 정보

nav : 다른 페이지 또는 현재 페이지의 다른 부분과 연결되는 네비게이션 링크(navigation links)들의 집합을 정의할 때 사용
> 일반적인 예로는 메뉴, 목차, 인덱스 등

section : HTML 문서에 포함된 독립적인 섹션(section)을 정의할 때 사용.
> 제목 요소(h1~h6)를 자식 요소로 포함하고 있는 경우가 대다수

footer : 태그는 문서나 특정 섹션(section)의 푸터(footer)를 정의할 때 사용
> <footer> 요소가 포함되어 있는 문서나 섹션에 대한 아래와 같은 정보를 포함  
    저자(author) 정보
    저작권 정보
    연락처
    사이트맵(sitemap)
    페이지 맨 위로 되돌아갈 수 있는 Top 버튼
    연관 페이지 등
